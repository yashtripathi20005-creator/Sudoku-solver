# Sudoku Solver and Game

A complete Sudoku game and solver application with a graphical user interface.

## Features

- **Play Mode**: Play Sudoku puzzles with interactive GUI
- **Puzzle Generation**: Generate puzzles with 4 difficulty levels
  - Easy: 30 cells removed
  - Medium: 45 cells removed
  - Hard: 52 cells removed
  - Expert: 58 cells removed
- **Solving Algorithms**:
  - Backtracking solver
  - Human-like strategies (naked singles, hidden singles)
  - Hybrid approach combining both methods
- **Game Features**:
  - Check puzzle validity
  - Get hints
  - Auto-solve with one click
  - Highlight cells
  - Visual feedback for errors
